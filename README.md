# GARCH-EVT-VaR_Reliance_Stock_Prediction
This project predicts Reliance Industries stock risk for 01-01-2026 using a hybrid GARCH–EVT framework. It models volatility clustering via GARCH(1,1), captures tail risk using Extreme Value Theory (POT–GPD), and estimates 99% VaR and Expected Shortfall with backtesting and visual diagnostics.
